 Assignment Overview

This repository contains C programs demonstrating the applications of Stack and Queue data structures.

The assignment focuses on solving real-world problems using these concepts and understanding their implementation in C.

 Programs Included
 1. Reverse a String using Stack
Reverses a given string using stack operations
Example: DATA → ATAD
 2. Balanced Parentheses
Checks whether an expression has balanced parentheses
Example: (A+B)*(C-D) → Balanced
 3. Next Greater Element
Finds the next greater element for each element in an array using stack

Example:

Input: 4 5 2 10 8  
Output:  
4 → 5  
5 → 10  
2 → 10  
10 → -1  
8 → -1  
 4. Printer Queue Simulation
Simulates a printer queue using queue operations
Features:
Add document
Print document
Display pending documents
 5. Circular Queue Operations
Implements circular queue using arrays
Operations included:
Enqueue
Dequeue
Peek
Display
 Technologies Used
Programming Language: C
Concepts: Stack, Queue, Circular Queue
